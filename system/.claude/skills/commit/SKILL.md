# Commit

Commit all current changes.

## Instructions

1. Run `git status` to see what's changed
2. Run `git diff` to review the changes
3. Stage all relevant modified and new files
4. Write a concise single-line commit message describing what changed
5. Commit and run `git status` to confirm

## Notes

- Do not push
- Do not amend existing commits
- Do not stage unrelated files
